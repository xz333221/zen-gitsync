# Product

## Register

product

## Users

- Developers who work with Git daily and want streamlined commit workflows
- Users who need to sync multiple folders/projects automatically
- DevOps engineers who orchestrate custom commands on a schedule
- Context: sitting at a desk, focused on development, often switching between coding and git operations

## Product Purpose

Zen GitSync is a git automation tool that removes friction from commit workflows. It provides:
- One-command git commits with smart defaults
- Automatic folder synchronization on intervals
- Custom command orchestration and scheduling
- A calm, non-intrusive GUI for visual git management

Success means: developers spend less time on git mechanics and more time coding.

## Brand Personality

- **Zen**: Minimal distraction, calm interface, no overwhelming options
- **Focused**: Every feature has a clear purpose; no bloat
- **Competent**: Professional tool for serious developers; not playful or toy-like

Voice: quiet confidence. No marketing speak. No exclamation marks. Direct and precise.

## Anti-references

- No rainbow gradients or "AI purple" aesthetic
- No cluttered dashboards with every metric visible at once
- No aggressive dark mode default — respect system preference
- No heavy shadows, glassmorphism, or decorative effects
- Avoid: Notion's casual friendliness, Vercel's gradient branding, Linear's saturated accents

## Design Principles

1. **Calm productivity** — The interface should never feel busy. Dense information is OK; cluttered is not.
2. **Opinionated defaults** — Sensible defaults reduce decision fatigue. Power users can still customize.
3. **Trust through clarity** — Show what's happening. git operations are visible and reversible.
4. **Performance is UX** — Instant feedback, no spinners unless necessary, lazy loading where appropriate.
5. **Quiet confidence** — Professional tool. Not flashy. Not boring. Just... works.

## Accessibility & Inclusion

- WCAG 2.1 AA compliance target
- Support system dark/light mode preference
- Reduced motion respected via `prefers-reduced-motion`
- Keyboard-navigable all primary actions
- Focus indicators visible and styled to match theme
